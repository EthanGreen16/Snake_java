# Snake Game in Java

A classic Snake game built with Java Swing. Control the snake to eat apples, grow longer, and avoid crashing into walls or yourself.

## Features

- **Smooth Movement**: Arrow key controls for snake movement
- **Progressive Difficulty**: Game speed increases as the snake grows
- **Score Tracking**: Real-time score display and final score on game over
- **Collision Detection**: Wall and self-collision detection
- **Restart Functionality**: Quick restart button after game over
- **Clean UI**: Styled game over messages and score display

## How to Run

### Prerequisites
- Java Development Kit (JDK) 8 or higher

### Running the Game
1. Clone the repository
2. Navigate to the project directory
3. Run the batch file:
